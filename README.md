# 𝖈𝖗𝖆𝖞𝖔𝖓𝖘 Gothic Sidebar

A dark, atmospheric chess.com club sidebar with gothic aesthetics and smooth animations.

## Features

✨ **Interactive Cathedral UI** - Immersive gothic layout with temple-like structure
🕯️ **Animated Candle Flicker** - Pure CSS inline candle lighting effects
🏔️ **Engraved Stone Text** - Carved lettering effect illusion
💫 **Hover Glow** - Candle flicker-inspired interactive glows
🌫️ **Fog Drifting Background** - Animated atmospheric mist
🎨 **Stained Glass Edges** - Glowing decorative borders
📋 **Event Board** - Links to club announcements and events
🎭 **Club Temple Homepage** - Full-featured gothic club interface

## Project Structure

```
gothicsidebar/
├── index.html          # Main sidebar markup
├── styles/
│   ├── main.css        # Core styles
│   ├── animations.css  # Animation effects
│   └── theme.css       # Gothic theme colors
├── scripts/
│   └── sidebar.js      # Interactive functionality
└── assets/
    └── images/         # Club imagery
```

## Setup

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required for basic version
